Starting from an empty folder, this video shows how to go from a research question ("how has U.S. homeownership changed over 50 years?") to a publication-quality figure in a single Claude Code session. Claude navigates the Census Bureau's housing data, downloads the right tables, and produces an R script with ggplot2. The key lessons: let Claude handle data acquisition, cite style guides you like (such as Kieran Healy's) instead of specifying every parameter, iterate in small steps, and always generate reproducible scripts rather than one-off outputs.
