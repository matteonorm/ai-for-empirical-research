Giving an agent broad permissions feels risky, but reflexively clicking "yes" on every prompt is not real safety either. This video reframes the question: the real lever is the environment, not the approval mode. It walks through Docker containers and devcontainers that mount only the folders you need, drop network access once setup is done, and let you run Claude in auto or YOLO mode inside a tight boundary. It also covers data governance (privacy filters for sensitive material, local models as a fallback) and long-running agent workflows like OpenClaw and the "Duncan Idaho" personal research agent.
